# Cashier Frontend

This is a minimal Vite + React frontend scaffold for the Cashier system.

Getting started (PowerShell on Windows):

```powershell
cd "f:\\new system\\Cashier\\frontend"
npm install
npm run dev
```

Open http://localhost:5173 in your browser. The dev server port may vary; check the terminal output.
